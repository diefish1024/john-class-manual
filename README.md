# John Class Manual

上海交通大学致远学院 John Hopcroft 班的非官方介绍手册，包含了日常生活、课内课程、科研 / 就业等信息与资源。

这个手册目前还在初步的建设阶段，欢迎想加入编写组 / 想做出贡献的同学联系我们！
